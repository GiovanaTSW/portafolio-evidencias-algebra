# Portafolio de Evidencias - Álgebra

Este repositorio contiene todas las actividades prácticas de HTML/CSS/JS realizadas durante el curso de matemáticas.

## Estructura del repositorio

- `index.html` — Página principal (menú/portafolio)
- `/css/style.css` — Hoja de estilos global
- `/js/script_global.js` — Script global si es necesario
- `/assets/images` — Imágenes para el portafolio
- `/actividades/actividadXX` — Cada actividad con `index.html` y `script.js`
